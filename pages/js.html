JavaScript - Weeks 3 & 4

JavaScript (or JS) is something we have spent quite a bit of time on and yet it is still something I have struggled with the most. This was, for most of us, our first step into a traditional programming language as HTML and CSS are not programming languages and so this was a challenge to overcome.

In the first lecture 13/04/23 we covered some of the basics and mainly focused on the maths application of JS. We learnt about var (variables) and also the more modern let/const name for variables. var is more of the traditional name for a variable but they are moving towards let (for a variable which can be change and the value overwritten) and const (for a variable which cannot be changed or edited).
