CSS (or Cascading Style Sheets) is what gives the bare bones of HTML its 'pizzazz'. It is used to make the webpage look more appealing and adds styles.

A few things which have tripped me up with CSS include remembering whether a certain selector is a class or an ID as using the incorrect one will not apply the desired style. Another thing which has given me trouble is the proprerties of flex...I have to sit and patiently go through the options until I find the one I want/need because I can never remeber fully how it works and which one I want.
